# https://gravatar.redirect.channel

Bounce requests to `gravatar.redirect.channel/your@email.com` to the corresponding `gravatar.com/avatar/hashed_email` URL.

All the magic happens in [docs/index.html](docs/index.html).

## Dependencies

* git v2+ `xcode-select --install`
* terraform 1.11+ `brew install terraform`

## Set Up

* `git clone git@github.com:beporter/gravatar.redirect.channel.git`
* `cd gravatar.redirect.channel`
* ``
