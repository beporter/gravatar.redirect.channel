# gravatar.redirect.channel

Bounce requests to `gravatar.redirect.channel/your@email.com` to the corresponding `gravatar.com/avatar/hashed_email` URL.

All the magic happens in [docs/index.html](docs/index.html).

## Development

* git
* Docker

Look in `bin/` for how to start up the various containers for testing locally.
